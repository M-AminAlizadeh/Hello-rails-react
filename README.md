## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Github account
- Git
- Vscode or any other IDE
- Ruby
- Rails
- Node.js
- Postgresql

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/M-AminAlizadeh/Hello-rails-react <directory name>
```

Setup database config from config/database.yml (change username and password)

Install

Install the dependencies:

```sh
  npm install
  bundle install
```

Create database:

```sh
  bin/rails db:create
```

### Usage

To run the project, execute the following command:

```sh
  ./bin/dev
```
